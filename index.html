IntroductionÂ¶

OpenClimateGIS is a Python package designed for geoprocessing and computation on CF-compliant climate datasets. It is “request-based” in its current release meaning an operation is entirely defined prior to execution. This approach allows the software to have nearly equivalent functionality from its Python and RESTful APIs.

Currently, there is a single point of entry to OpenClimateGIS: the OcgOperations object. This documentation will describe the Python syntax as well as the equivalent RESTful form of its argumentation.

There is additional project content for OpenClimateGIS hosted on it’s CoG Site. Please visit the Contact Information page for contact points.
